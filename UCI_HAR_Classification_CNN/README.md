# Residual Block을 활용한 CNN 분류 개선
## 1. 문제점
데이터를 분류하는 과정에서 특히 **Sitting과 Standing** 클래스의 센서 신호 패턴이 매우 유사하여 분류가 어려워 정확도가 96% 이상 나오지 않았다.

이 두 클래스는 모두 정지 상태에 가까운 신호를 가지며 큰 움직임이 아닌 중력 방향이나 자세 변화와 같은 미세한 차이로 구분된다.

따라서 이 두 클래스를 분류할 때는 전체적인 패턴보다는 작은 특징 차이를 정확하게 분류하는 것이 중요한 문제였다.

그래서 Sitting과 Standing의 작은 특징 차이를 효과적으로 학습하기 위해 입력을 유지하고 변화량만 학습하는 Residual Block을 적용하였다.

## 2. CNN만 활용했을 때의 한계점
- 작은 특징을 파악하기 위해 깊은 모델을 구성할 경우 기울기 소실 문제가 발생하여 학습이 어려워진다.
- 층이 깊어질수록 입력 정보가 점차 변형되면서 중요한 특징이 손실될 가능성이 존재한다.
- 모델이 전체 패턴을 만드는 과정에서 작은 차이는 무시될 수 있다.

## 3. Residual Block의 구조
Residual Block은 입력을 두 가지 경로로 전달하는 구조

Identity 경로 : 입력 x를 그대로 전달

Residual 경로 : Conv, ReLU 등의 레이어를 통해 F(x)를 학습

두 가지 경로를 통해 나온 벡터값 입력 x와 변화량 F(x)를 element-wise로 더해서(같은 위치에 있는 값끼리 1:1로 더한다) y를 출력한다.

-> y= x + F(x)

## 4. 동작 방식
### 일반 CNN
1) 입력 x가 들어온다.
2) Conv -> ReLU -> Conv 등 연산을 통해 새로운 값 F(x)을 생성
3) 이 값이 최종 출력값이 된다.
4) 최종 출력값을 기반으로 역전파를 수행하여 가중치를 업데이트하여 F(x)가 정답에 맞도록 학습한다.
   
-> 모델이 출력값 자체를 생성 해야한다.

예) 입력이 100일 때 정답이 103이라면 모델이 직접 103을 만들어야 한다.

### Residual Block 기반의 ResNet 구조
1) 입력 x가 들어온다.
2) 입력은 두 경로로 나뉜다.
	2-1) 입력을 그대로 전달하는 경로
	2-2) Conv -> ReLU -> Conv 등 연산을 통해 변화량 F(x)을 생성
      - 이때 Residual Block에서는 입력 x와 F(x)를 더해야 하기 때문에 두 값의 크기를 동일하게 하기 위해서 Conv 연산 시 padding, stride, 채널 수 등을 조절하여 동일한 형태가 되도록 맞춘다.
3) 두 값을 더하여 최종 출력 생성
4) 최종 출력값과 정답의 차이를 기반으로 역전파를 수행하여 가중치를 업데이트하여 변화량 F(x)가 정답에 맞도록 학습한다.
   
-> 모델이 출력값 자체를 생성하는 것이 아닌 입력과 출력의 차이(변화량)을 학습한다.
   
예) 입력이 100일 때 정답이 103이라면 모델은 입력과 출력의 차이 값인 변화량 +3을 학습하면 된다.

일반 CNN : y = F(x)
Residual Block 기반의 ResNet 구조 : y = x+F(x)

**본질적인 차이 : 일반 CNN은 전체 값을 새로 만드는 구조, Residual 구조는 기존 입력 값에 작은 변화량만 더하는 방식**

## 5. Residual 구조의 장점
### 1. 학습 난이도가 감소  
Residual 구조의 핵심은 모델이 출력값 자체를 만드는 것이 아니라 입력을 얼마나 수정해야 정답이 되는지 변화량만 학습을 하면 된다.

만약 입력과 정답이 유사한 경우 F(x)가 0에 가까운 값이 되기 때문에 큰 값을 찾는 문제에서는 작은 값을 찾는 문제로 학습을 진행할 수 있어 효율적인 학습을 할 수 있게 된다.

또한 입력과 출력 값이 같다면 F(x) = 0이 되기 때문에 입력이 그대로 출력으로 전달되므로 모델이 아무런 학습을 진행하지 않아도 최소한의 성능을 유지할 수 있게 된다.

### 2. 기울기 소실 개선
여러 레이어를 거치면서 역전파가 진행되는 과정에서는 작은 값이 반복적으로 곱해지기 때문에 기울기 소실 문제가 발생할 수 있다. 

하지만 Residual 구조에서는 출력이 y = x+F(x) 로 입력이 그대로 전달되는 identity 경로로 인해 미분값에 항상 1이 포함되므로 기울기가 완전히 사라지지 않고 안정적으로 전달된다.

### 3. 정보 보존
입력 x가 출력에 직접 포함되기 때문에 중요한 특징 정보가 레이어를 거치면서 사라지지 않게 된다.

## 6. HAR 데이터에서 Residual 구조의 효과
HAR 데이터에서 Sitting과 Standing 클래스는 전체적인 신호 패턴이 매우 유사하여 중력이나 자세 변화와 같은 미세한 차이로 구분된다. 

따라서 모델은 전체적인 패턴을 새롭게 학습하기 보다는 기존 신호에서 나타나는 작은 차이를 정확하게 포착하는 것이 중요하게 된다.

처음 성능을 테스트한 일반 CNN 구조에서는 여러 레이어를 거치면서 입력 데이터는 계속 변형이 되기 때문에 중요한 특징이 손실되거나 전체 패턴을 만드는 과정에서 Sitting과 Standing 간의 작은 차이는 무시되었다.

반면 Residual 구조에서는 입력이 그대로 전달되면서 변화량만 학습할 수 있기 때문에 기존 값에서 미세한 차이만을 파악하여 학습을 진행할 수 있었다.

이러한 구조로 인해 일반 CNN보다 Residual 구조는 Sitting과 Standing 처럼 **미세한 클래스 간의 차이를 구분해야 하는 문제에서 더 효과적인 구조**임을 알 수 있었다.
