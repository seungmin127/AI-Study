# Multichannel CNN-GRU 기반 HAR Dataset 분류
## 참고 논문
본 모델은 A Multichannel CNN-GRU Model for Human Activity Recognition의 제안 모델을 기반으로 구현하였다.

## 모델 구성
현재 모델은 Multichannel CNN + GRU + GAP 구조로 이루어져 있다.

## 단일 CNN과 Multichannel CNN 비교
### 단일 CNN
- 하나의 그림을 한 사람이 보는 구조.
- 한 사람은 하나의 기준으로만 그림을 보기 때문에 세부적인 부분을 잘 볼 수도 있지만 전체 흐름을 놓치는 가능성 존재
- 반대로 전체적인 부분을 보다가 작은 특징을 놓치는 가능성이 존재.

### Multichannel CNN
- 하나의 그림을 여러 사람이 보는 구조.
- 각 사람은 서로 다른 기준으로 그림을 보기 때문에 누군가는 가까이서 보고 다른 누군가는 떨어져서 보고 또 다른 누군가는 전체를 한 번에 봄.
- 여러 사람이 본 데이터들을 하나로 결합.

이러한 구조적 특징을 통해 Multichannel CNN은 다양한 범위의 특징을 동시에 추출할 수 있어 단일 CNN보다 더 풍부한 정보를 기반으로 판단을 할 수 있다.

따라서 HAR 같은 시계열 데이터에서 특징 표현 능력을 강화하여 분류 정확도를 향상 시킬 수 있다.

## LSTM과 GRU 비교
### LSTM
- 게이트 : 3개 (Forget, Input, Output)
- 이전 정보(벡터)를 기억하는 값 : 2개 (Cell, Hidden)

특징 : 메모리를 두 층으로 관리
	
	- 정보 장기 의존성에 강함
	
	- 구조가 복잡하고 파라미터 수가 많음

### GRU
- 게이트 : 2개 (Update, Reset)
- 이전 정보(벡터)를 기억하는 값 : 1개 (Hidden)

특징 : 메모리를 하나로 통합
	
	- LSTM에 비해 정보 흐름이 하나이므로 구조가 단순함
	
	- 계산량과 파라미터 수가 적음

이러한 특성을 정리하면 LSTM은 복잡하지만 표현력이 강하고 GRU는 단순하고 효율적이다.

### HAR에서 GRU를 사용한 이유
1. HAR 데이터가 길지만 그렇게 길지가 않다.
- 보통 128timestep의 데이터 약 2~3초, 걷기-앉기 같은 짧은 패턴이기에 초장기 의존성이 아니기 때문에 굳이 LSTM의 복잡한 메모리 구조가 필요없다.
2. 파라미터 수가 적다.
- GRU는 LSTM보다 게이트 수가 적어 연산에 필요한 가중치가 줄어들기 때문에 파리미터 수가 감소하여 학습 속도와 효율성이 높아진다.

  논문에서의 핵심 문장 : "GRU has a smaller number of parameters" (GRU는 더 적은 수의 파라미터를 가진다.)
3. 노이즈 처리
- 센서 데이터는 노이즈가 많기 때문에 GRU는 Updata Gate로 이전 흐름과 비교하여 신뢰 가능한 정보만 반영할 수 있다.

  논문에서의 핵심 문장 : "GRU enables fault-tolerant capability" (GRU는 오류(노이즈)에 대해 견딜 수 있는 능력을 가진다.)

### CNN과 GRU 결합 이유
CNN은 다양한 범위의 특징을 동시에 추출할 수 있지만 시간의 흐름을 반영하지 못하고 노이즈 데이터에 취약한 한계를 가지고 있다. 

이를 해결하기 위해 GRU와 결합하여 시간적 흐름을 파악하여 학습할 수 있고 노이즈 제거 및 정보 선택을 할 수 있다.

따라서 CNN과 GRU 결합 시 정확도와 효율성, 안정성을 확보할 수 있다.

### GAP (Global Average Pooling)를 사용한 이유
기존 Fully Connected Layer만 사용했을 때 모든 값을 하나하나 연결해서 최종 분류를 하기 때문에 파라미터 수가 많아 과적합이 발생하기가 쉽다.

반면 GAP는 각 특징 맵을 하나의 값으로 평균을 내어 중요한 특징을 유지하면서 불필요한 계산을 줄일 수 있다.

최종 분류를 위해 GAP를 사용하더라도 DENSE 층이 필요하지만 하나의 층만을 사용하여 파라미터 수를 줄일 수 있다. 

따라서 GAP를 사용하면 모델의 파라미터가 적어지기 때문에 과적합이 줄어 학습이 안정적으로 이루어진다.
